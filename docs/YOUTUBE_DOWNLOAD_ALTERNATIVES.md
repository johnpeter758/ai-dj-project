# YouTube Download Alternatives

## Current Status
- yt-dlp is blocked for specific videos on your IP
- Need different IP or authentication

## Workarounds Found

### 1. Browser Extension (RECOMMENDED)
- **4K Video Downloader** - Browser extension that works
- **YouTube Video Downloader** extension
- Download manually, copy to Music folder

### 2. Web Services (No Install)
- **y2mate.com** - Paste URL, download MP3
- **ytmp3.net** - Simple YouTube to MP3
- **onlinevideoconverter.com**

### 3. VPN Change
- Switch Proton VPN to different country
- Some countries work better (Netherlands, Japan)

### 4. Different Account
- Create new YouTube account
- Or wait 1-2 weeks for block to lift

### 5. Use Spotify (if you have Premium)
- spotdl can download from Spotify
- Works even when YouTube is blocked

### 6. Local YouTube Videos
- If you have videos in other apps, import them
- Use AnyTrans or iMazing to extract
