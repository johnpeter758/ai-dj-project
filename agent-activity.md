# Agent Activity Channel

**Channel ID:** 1479544840894025809
**Channel Name:** agent-activity

## Rule
When deploying sub-agents, always post status updates to this channel.

### Format
```
🔄 [Agent Name] started - [Task description]
✅ [Agent Name] completed - [Result summary]
❌ [Agent Name] failed - [Error details]
```

### Examples
```
🔄 Research agents launched - Covering Twitter, Reddit, HN, YouTube, Blogs
✅ Research complete - 5 sources analyzed, report saved
```
