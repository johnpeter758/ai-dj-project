# Sidechain Compression

## Basic Setup
- Kick → Compressor Sidechain Input
- Bass/Synths → Compressor Input → Output

## EDM Settings
- Ratio: 4:1 to 8:1
- Attack: 1-10ms
- Release: 50-200ms
- Threshold: 3-6dB gain reduction

## Ghost Kick
- Create synthetic kick for sidechain only
- Mute the track
- Automate for unique rhythms

## Parallel Sidechain
- Ratio: 10:1 or infinite
- Attack: 0.1-1ms
- Blend: 30-70% wet

## Mid/Side
- Side: Glue stereo width
- Mid: Control bass
