# Song Arrangement AI

## Structure Templates

### Pop
Intro (8) → Verse (16) → Pre-Chorus (8) → Chorus (16) → Verse (16) → Pre-Chorus (8) → Chorus (16) → Bridge (8) → Chorus (16) → Outro (8)

### EDM
Intro (8) → Build (8) → Drop (16) → Build (8) → Drop (16) → Outro (8)

### Hip-Hop
Intro (4) → Verse (16) → Hook (8) → Verse (16) → Hook (8) → Outro (8)

## Energy Curve
- 0.0 = minimal
- 0.3 = verse
- 0.5 = pre-chorus
- 0.8-1.0 = chorus/drop

## Transition Types
- HARD_CUT: Instant change
- FILL_BEAN: Drum fill
- RISER: Sweeping riser 4-8 bars
- SWELL: Volume swell → drop
- FILTER_SWEEP: Low-pass sweep

## Timing (BPM=120)
- Intro: 8-16 bars (8-16 sec)
- Verse: 16-32 bars
- Pre-Chorus: 4-8 bars
- Chorus: 8-16 bars
- Drop: 8-16 bars
- Outro: 8-16 bars
