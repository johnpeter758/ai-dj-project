# Demucs Setup

## Status: ✅ Ready

## Installation
- Demucs v4.0.1 installed
- ffmpeg v8.0.1 installed

## Usage
```bash
# 2 stems (vocals)
demucs --two-stems=vocals track.mp3

# 4 stems
demucs track.mp3
```

Created: `stem_separator.py`
