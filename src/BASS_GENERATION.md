# Bass Generation

## Synthesis Types
- **Sub bass**: Sine 20-60Hz
- **Synth bass**: Saw/square + LP filter
- **808 bass**: Pitch drop, long release, saturation

## Genre Patterns
- **Hip-hop**: Chromatic runs, 808 slides
- **House**: Root/fifth on 8ths
- **Dubstep**: Staccato wobble
- **R&B**: Walking bass

## Melody Complementation
- Use chord tones
- Keep below 250Hz
- Contrary motion

## Groove
- Note placement varies by genre
- Ghost notes (10-30% velocity)
- Swing: 0-66%

## 808 Synthesis
- Sine + pitch envelope drop
- Root → octave below
- Distortion
