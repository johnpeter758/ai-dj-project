# DAW Mixing Techniques

## Hip-Hop
- Layer kick + 808 (tuned to root note)
- High-pass everything except bass/kick at 60-80Hz
- Parallel compression on snare
- Vocals: de-ess, compress 2-3dB, send to reverb/delay

## EDM
- Sub bass: mono below 100Hz, stereo above
- Sidechain pads to leads
- Limit master: -0.5 to -1dB True Peak
- -14 LUFS integrated for streaming

## Bus Processing

### Drum Buss
- Compressor: 4:1, 10-30ms attack, 50-100ms release
- EQ: HPF 30Hz, subtle low-mid boost

### Bass Buss
- Fast attack (1-5ms), 4:1 ratio
- High-pass 25Hz

## Parallel Compression
- Ratio: 8:1 to 20:1
- Attack: 1-3ms, Release: 50-100ms
- Blend 20-40% wet for punch

## Mid/Side
- Boost sides on leads above 2kHz (+2-4dB)
- Cut sides below 100-150Hz on bass (keep sub mono)
- Cut 300-600Hz in sides for clarity

## Reference Track
- Match loudness (LUFS)
- A/B every 30-60 seconds
- Check: sub bass, vocal, stereo width, highs
