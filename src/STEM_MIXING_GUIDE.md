# Stem-Based Mixing Guide

## 1. Key Principles
- Match tempo and key
- Phase alignment
- Gain staging (-14dB LUFS)
- EQ matching

## 2. Demucs Tips
- Best model: `htdemucs_ft`
- GPU for speed
- Post-process with noise gates

## 3. Drum-Bass Layering
- Align kick to downbeat
- Sidechain compression
- EQ match tonal character

## 4. Vocal Processing
- De-ess aggressively
- Light noise gate
- Compress gently (2-3dB)
- Chain: De-esser → EQ → Compressor → Reverb → Delay → Saturation

## 5. Seamless Blending
- High-pass/low-pass to create space
- Match reverb tails
- Bus compression for "glue"
